# Google ADK

Purpose: target live agent runtime for VetAgent when `AGENT_RUNTIME=google-adk`.

Runtime package:

- npm: `@google/adk`
- local declaration: `packages/agents/package.json`
- installed by root `package-lock.json`
- upstream source: `https://github.com/google/adk-js`
- license: Apache-2.0
- checked version: `1.1.0`

Why the code is not vendored here:

- The app should consume ADK as an npm package, not a copied upstream repo.
- `opensrc/google-adk/` is the provenance folder so the source is obvious.
- `node_modules/@google/adk` is generated by `npm install` and should not be committed.

Runtime env:

- `AGENT_RUNTIME=google-adk`
- `GEMINI_API_KEY` or `GOOGLE_API_KEY`
- for Vertex-backed ADK: `GOOGLE_GENAI_USE_VERTEXAI`, `GOOGLE_CLOUD_PROJECT`, `GOOGLE_CLOUD_LOCATION`

Codex skills used:

- Google ADK lifecycle skills: `google/agents-cli`
- TypeScript ADK skill: `vysotin/cc_google_adk_skill`

Those skills are local Codex guidance, not app runtime code.
